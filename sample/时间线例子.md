# 时间线格式示例

以下为记忆系统从章节中提取的核心事件格式参考。多个事件之间用空行分隔。

---

事件：云岭关灵脉异动初查

地点：云岭关北侧哨塔

参与者：林岚, 哨塔守将韩渡

过程：林岚抵达云岭关，发现北侧灵脉读数异常升高。韩渡汇报近三日夜间有异响，但未发现实体入侵。林岚决定次日深入裂隙边缘勘察。

结局状态：林岚驻扎云岭关，已锁定异常区域，尚未接触裂隙生物。

事件：废墟遭遇裂隙游民

地点：云岭关外三里·旧驿道废墟

参与者：林岚, 裂隙游民小队首领铁藜

过程：林岚勘察途中遭遇游民小队，双方短暂对峙。铁藜声称只是在采集遗物，并暗示「有人在动灵脉枢纽」。林岚未动手，换取了一条关于残月会的模糊线索。

结局状态：林岚获得残月会可能介入的情报，继续向灵脉异常点推进。

事件：盟议堂紧急通报

地点：苍梧城·盟议堂

参与者：沈清河, 林岚, 各城代表

过程：林岚通过传讯器回报情况，沈清河下令暂缓云岭关民用灵材输送，并派援兵支援。会上望海埠代表质疑残月会传闻缺乏实证，盟主决定给林岚七日调查期限。

结局状态：林岚获七日专权调查期，云岭关进入戒备状态，援兵两日后抵达。

## 字段说明

| 字段 | 说明 |
|------|------|
| 事件 | 本章核心事件标题，简明概括 |
| 地点 | 事件发生的主要地点 |
| 参与者 | 涉及角色，逗号分隔 |
| 过程 | 事件经过的简要描述 |
| 结局状态 | 本章结束时相关角色或局势的状态 |

> 说明：系统还会为每条时间线生成实体标签与向量嵌入，用于后续检索，无需在示例中手写。
